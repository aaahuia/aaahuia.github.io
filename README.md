# Donghui's Blog

个人博客，记录生活、旅行与思考。

## 在线访问

[wangdonghui.com](https://wangdonghui.com)

## 项目结构

```
├── index.html          # 首页（文章列表）
├── posts/              # 文章页面
├── css/style.css       # 样式
├── images/             # 配图
└── CNAME               # 自定义域名配置
```

## 技术栈

纯静态站点，HTML + CSS，通过 GitHub Pages 部署。

## 本地预览

用任意静态服务器打开即可，例如：

```bash
npx serve .
```

然后访问 `http://localhost:3000`。
